ATIVIDADE DE HOJE

1- APAGUE O REPOSITÓRIO AULAS_AGOSTO
2- RECRIE O REPOSITÓRIO AULAS_AGOSTO
3- ORGANIZANDO SEUS DIRETÓRIOS
- VOCÊ TEM UMA PASTA DA DISCIPLINA (TETEC)?
 - SE A RESPOSTA É SIM ENTÃO DENTRO DESSA PASTA CRIE UMA PASTA CHAMADA AULA2_FLEX
 - SE A RESPOTA É NÃO, ENTÃO CRIE A PASTA TETEC EM SEGUIDA CRIE A PASTA AULA2_FLEX

4-CHAME O TERMINAL (GIT BASH) A PARTIR DA PASTA TETEC
5-INICIALIZE O REPOSITÓRIO LOCAL (git init)
6- CONFIGURE AS CREDENCIAIS (USER.NAME E USER.EMAIL)
7- ACESSE O VSCODE E CRIE UM ARQUIVO INDEX.HTML DENTRO DA PASTA AULA2_FLEX
8- EMPURRE AS ALTERAÇÕES PARA O SEU REPOSITÓRIO REMOTO (
	git add . 
	git commit -m "criando pasta de trabalho e arq. index"
	git remote add origin <link do seu repositório>
	git push origin master
